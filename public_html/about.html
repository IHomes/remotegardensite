About this project


MyGardenSpy is the 'front end' of a (Raspberry Pi) garden weather station operating miles from my home, fueled by solar, and connected over RF (with a DNT900), using the rpi900 interface board.

Details on the build, design and all related hardware and software can be found on my blog. The source code and setup notes also available on GitHub. For occasional updates on this project or others, follow my Twitter.
